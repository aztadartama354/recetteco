<h3>Ingrédients de croque de gauffres salées :</h3>

<ol>
	
		<li>Vous devez vous préparer 300 g farine intégrale. </li>
	
		<li>Vous avez besoin 50 g beurre fondu. </li>
	
		<li>Utilisation 400 ml lait. </li>
	
		<li>Vous devez vous préparer 1 sachet de levure. </li>
	
		<li>Vous avez besoin 2 oeufs. </li>
	
		<li>Vous avez besoin  Sel. </li>
	
		<li>Vous avez besoin  Poivre. </li>
	
</ol>

<h3>Croque de gauffres salées instructions :</h3>

<ol>
	
		<li>
			Pour la pâte :faites fondre le beurre et mélangez tout les ingrédients avec dans un plat. laissez reposer au frigo pendant 1h..
			
			
		</li>

			Votre pâte est prête. Préchauffez le four a 200 degrés..
			
			
		</li>
	
		<li>
			Faites cuire votre pâte dans le gaufrier. Ensuite garnissez de votre choix. mettez cuire dans le four..
			
			
		</li>
